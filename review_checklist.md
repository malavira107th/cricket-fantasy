# Website Review Checklist

## Business Model Verification
- [ ] All pages clearly state "FREE TO PLAY"
- [ ] No mention of real money prizes or winnings
- [ ] No payment/withdrawal features
- [ ] Consistent messaging about "bragging rights only"
- [ ] Legal disclaimers present on all relevant pages

## Pages to Review

### Main Pages
- [ ] Home - Check hero section, CTAs, features
- [ ] Signup - Verify no payment required
- [ ] Login - Standard login flow
- [ ] Dashboard - Check for money-related features
- [ ] Leaderboard - Ensure no prize mentions

### Contest Pages
- [ ] Contests - Check contest cards for prize info
- [ ] Contest Details - Verify entry is free
- [ ] Create Team - No payment required
- [ ] My Teams - Display only
- [ ] My Contests - No withdrawal options

### Information Pages
- [ ] How to Play - Clear free-to-play explanation
- [ ] FAQ - Answer money-related questions correctly
- [ ] About - Company info accurate

### Legal Pages
- [ ] Terms - Correct business model described
- [ ] Privacy - Standard privacy policy
- [ ] Responsible Gaming - Appropriate for free platform
- [ ] Contact - Contact info correct

### User Pages
- [ ] Profile - No wallet/payment sections
